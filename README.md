# PS2 Texture Sorter

This project aims to provide a tool for sorting textures used in PS2 games. The goal is to streamline the process of managing and utilizing textures, ensuring efficient performance and organization.

## Features
- Easy texture sorting
- User-friendly interface
- Compatibility with multiple formats
